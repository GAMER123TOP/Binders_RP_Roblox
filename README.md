# Roblox Macros для РП

AutoHotkey скрипты для упрощения РП-отыгровок в Roblox.

![Version](https://img.shields.io/badge/version-1.5-blue)
![Platform](https://img.shields.io/badge/platform-Windows-0078D6)
![License](https://img.shields.io/badge/license-MIT-green)

---

## 🚀 Установка через лаунчер

1. Скачай лаунчер по ссылке: [ССЫЛКА]
2. Запусти лаунчер → скрипты автоматически скачаются и обновятся
3. Выбери нужный скрипт и запусти

> 💡 Лаунчер сам обновляет скрипты до последней версии — ничего качать вручную не нужно.
