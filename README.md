# MVD Macros для Roblox

AutoHotkey скрипт для упрощения РП-отыгровок в Roblox (МВД).

---

## 📥 Установка

1. Скачай и установи [AutoHotkey v2](https://www.autohotkey.com/)
2. Скачай файл `MVD_Macros.ahk` из этого репозитория
3. Запусти файл двойным щелчком

---

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Numpad1` | Чистка ПМ |
| `Numpad2` | Представление |
| `Numpad3` | Ксива |
| `Numpad4` | Наручники / Обыск |
| `Numpad5` | Рация |
| `Numpad6` | Миранда |
| `Numpad7` | Арест |
| `Numpad8` | ПМ в боевое |
| `Numpad9` | Мегафон |
| `Ctrl+Numpad4` | Настройки (смена клавиши чата) |
| `Ctrl+Esc` | Остановка скрипта |
| `Ctrl+R` | Перезапуск скрипта |
