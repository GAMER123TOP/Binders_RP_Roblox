# Roblox Macros для РП

AutoHotkey скрипты для упрощения РП-отыгровок в Roblox.

---

## 📥 Установка

1. Скачай и установи [AutoHotkey v2](https://www.autohotkey.com/)
2. Выбери свою игру и скачай нужные файлы:
   - **`MVD_Macros.ahk`** — для сотрудников МВД/ГИБДД
   - **`Civilian_Macros.ahk`** — для гражданских (подходит для всех игр)
3. Запусти файл двойным щелчком

---

## 🎮 Игры

| **Зареченск** | Основная игра, на которой базируются все скрипты |

| **Кемерово** | Адаптированная версия для сервера Кемерово |

| **Провинция** | Адаптированная версия для сервера Провинция |

---

## ⌨️ Горячие клавиши

Все горячие клавиши отображаются в **оверлее** при запуске скрипта.
Просто запусти нужный файл и увидишь список всех клавиш.

> 💡 Оверлей можно перетаскивать за любую область.
